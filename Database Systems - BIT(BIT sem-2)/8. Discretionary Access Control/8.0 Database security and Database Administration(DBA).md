- To protect database against threats, 4 measures can be implemented,
1. Access Control
2. Inference Control
3. Flow control
4. Encryption

- DBA is the central authority for managing a database system. DBA's responsibilities include granting privileges to users who need to use the system and classifying users and data in accordance with the policy of the organization. 
DBA is privileged for performing,
1. Account creation
2. Privilege revocation
3. Security level assignment
## Access Control
DBMS offers two main approached to access control,
1. Discretionary access control
2. Mandatory access control


#### 2. Mandatory Access Control
Govern the access to the information by the individuals on the basis of the classification of subjects and objects in the system.
